El codigo funciona mediante Javascripts para calcular listas de gastos y ganancias.
con esto puedes resolver un problema de cuanto bas a gastar exactamente, se permiten grandes cantidades.
el balance se actualiza si elimina la lista para poder ordenarse.

Solo coloca un objecto, luego montón: de cuanto cuesta el objecto que coloco, y en la pestaña siguiente es gasto o ganancia

ganancia: lo que tiene como dinero o salario

gastos: el objecto que cuesta exactamente y lo que se gastara con el
